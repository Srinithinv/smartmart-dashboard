SmartMart Admin Dashboard
A modern React-based Admin Dashboard UI for managing orders, customers and store settings.

Features
Fake Login Authentication (localStorage)
Sales Analytics Chart (Recharts)
Orders Management with Filters
Customer Cards with Search & Sort
Settings Panel with Toggles
Responsive UI Design
Glassmorphism + Gradient UI

Tech Stack
React.js
React Router
Tailwind CSS
Recharts
LocalStorage (Fake Auth)
